# Pomodoro Timer API

Backend-сервис для приложения-таймера Pomodoro. Обеспечивает управление пользователями, задачами, сессиями таймера и сбор статистики.

## Tech Stack

- **Framework:** FastAPI (async)
- **Database:** PostgreSQL
- **ORM:** SQLAlchemy 2.0+ (async)
- **Migrations:** Alembic
- **Validation:** Pydantic
